# Contributing

Thanks for helping improve `secretlint-config-nick2bad4u`.

## Development

1. Install dependencies with `npm install`.
2. Make changes to `.secretlintrc.json`, `preset.mjs`, `index.d.ts`, or docs.
3. Run `npm run verify` before opening a pull request.

## Pull requests

- Keep changes focused.
- Explain why Secretlint rule or dependency changes are needed.
- Include consumer migration notes when changing the exported config shape.
